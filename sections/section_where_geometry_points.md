## Where the geometry points

The persistent subspace is a descriptive finding. I have shown that it exists, that it is low-dimensional, and that it depends on sequential context rather than token identity. What I have not shown is that it is causally upstream of anything the model does. That caveat matters for how strongly to read the following, which are directions the geometry suggests rather than conclusions the experiment supports.

**Implication 1: Surgical activation steering.** Current activation steering interventions add a vector to the residual stream at a given token position without distinguishing between directions that carry long-timescale state and directions that are transient. The geometry identified here suggests a decomposition: any intervention can be projected onto the persistent subspace and its orthogonal complement separately. Steering the transient component would change what the model is processing at the current position; steering the state-bearing component would change what it carries forward across subsequent tokens. Whether this decomposition produces meaningfully different behavioral effects is an open question, but it is the natural next experiment. Work on manifold steering, such as what Goodfire and others are pursuing, would benefit from knowing which directions in the residual stream are durable enough to be worth targeting.

**Implication 2: Parameter decomposition by temporal role.** Once we have a candidate persistent subspace $P \subset \mathbb{R}^{d_{\mathrm{model}}}$, any component that writes into the residual stream can be split into persistent and transient parts: $PW$ and $(I-P)W$. For an attention output matrix $W_O$ or MLP output matrix $W_{\mathrm{out}}$, a simple score is
$$
\mathrm{persistent\_write\_fraction}(W)=
\frac{\|PW\|_F^2}{\|W\|_F^2}.
$$
Read matrices can be scored analogously by $\|WP\|_F^2 / \|W\|_F^2$. This gives each head or MLP a temporal profile: does it mainly read and write durable cross-token state, or fast-decaying local computation? The connection to VPD-style parameter decomposition is useful context but not exact. This is not a learned decomposition of parameters by optimization; it is a geometric projection of existing weights through a discovered residual-stream subspace.

**Implication 3: The KV cache.** The residual-stream result says where long-timescale state may live, but not how that state moves between token positions. Attention is the transport mechanism: previous positions store keys and values, and later positions read a weighted combination of them. If the persistent subspace carries long-timescale state, then positions with large persistent residual energy, $\|Pr_s\|^2$, should be more likely to attract long-range attention, for example high $\sum_{t>s+k} A_{t,s}$. The same question can be asked in key/value space: are cached states with stronger persistent-subspace content more likely to be routed forward? This differs from most KV-cache work, which is usually about efficiency, compression, or position selection. The question here is whether residual geometry predicts which cached token states are worth transporting across context.

**Implication 4: A temporal dimension for mechanistic interpretability.** The second wave of mech interp, organized around superposition, sparse autoencoders, and the linear representation hypothesis, is largely a static and per-position picture. It asks what the model represents at a given token in a given layer. The persistent subspace introduces a different axis: not what is represented, but how long it lasts. Some directions are fast and some are slow, and that distinction is geometrically structured. Whether this is the right frame for the kinds of dynamic, computation-oriented questions that characterize the emerging third wave of the field is not something this pilot experiment can settle. But the temporal structure of the residual stream is almost entirely unstudied, and the methods here, particularly the time-lagged covariance objective, suggest a concrete way to study it.

The caveat that runs through all of this is causality. The 31 directions persist, but persistence alone does not establish that the model uses them. It is possible that they are a byproduct of some other mechanism and that ablating them would have little effect on behavior. Establishing a causal role is the work I was not able to do here, and it is the work that would convert these directions from a geometric curiosity into an interpretability tool.
